# Candy Language Support (VS Code / Cursor)

This extension provides:

- Language registration for `.candy` and `.cdy`
- Syntax highlighting for core Candy syntax
- Bracket/comment/auto-close behavior

## Build VSIX

From this folder:

```bash
npm install
npm run package
```

Output:

- `candy-language-0.1.0.vsix`

## Install VSIX

### VS Code

- Extensions panel -> `...` -> `Install from VSIX...`

### Cursor

- Extensions panel -> `...` -> `Install from VSIX...`
